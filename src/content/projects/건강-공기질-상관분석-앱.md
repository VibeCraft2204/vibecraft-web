## 📖 상세 내용
프로젝트 개요
간이 미세먼지 측정기를 모바일 앱과 연동하여 수집하고, 행동 및 식이 일지를 작성하게 하여 산모와 아이의 건강과 생활 패턴 및 미세먼지 간 상관관계를 분석하기 위한 연구과제
주요 목적
- 산모/아이 건강과 미세먼지 상관관계 연구
- BLE 기반 미세먼지 측정기 연동
- 행동/식이 일지 데이터 수집
주요 기능
- BLE 통신 프로토콜 (청크 48 bytes)
- 4개 소셜 로그인 (Google, Apple, Kakao, Naver)
- Firebase Auth 통합
- Provider 상태 관리
프로젝트 성과
- BLE 파일 전송 속도 3배 향상 (청크 16→48 bytes)
- 4개 플랫폼 통합 로그인으로 회원가입 장벽 제거
- Flutter 3.22 마이그레이션 완료
## 🛠️ 사용 기술 및 라이브러리
- F/E: Flutter 3.22, Dart, Provider
- B/E: FastAPI
- DB: PostgreSQL
- IoT: BLE (Bluetooth Low Energy)
- Tool: Firebase Auth, Firebase Analytics
