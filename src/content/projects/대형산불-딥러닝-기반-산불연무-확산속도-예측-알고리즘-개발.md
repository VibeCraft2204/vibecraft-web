## 📖 상세 내용
프로젝트 개요
산불 및 연무 확산을 예측하며 피해평가까지 사후 제공가능한 대형 산불의 피해를 최대한 빠르게 줄이고 피해 예측을 위한 대형산불 통합 플랫폼 구축 연구과제로 정부 30대 연구과제로 선정된 대규모 컨소시엄으로 구성되어 진행되는 프로젝트
주요 목적
- 기상청 API 연동 대규모 기상 데이터 수집
- 산불연무 확산 예측을 위한 데이터 파이프라인 구축
- 안정적인 대용량 데이터 수집 시스템 개발
주요 기능
- FastAPI 기반 RESTful API 서버
- 기상청 API 연동 8개 기상 요소 동시 수집
- Rate Limiting 시스템으로 API 안정성 확보
- 비동기 병렬 처리 (aiohttp) 기반 고속 데이터 수집
- 중복 데이터 필터링 및 스토리지 최적화
프로젝트 성과
- API 요청 실패율 95% 감소 (Rate Limiting)
- 데이터 수집 속도 3배 향상 (비동기 병렬 처리)
- 디스크 사용량 60% 절감 (중복 저장 방지)
## 🛠️ 사용 기술 및 라이브러리
- B/E: FastAPI, aiohttp, Pydantic
- DB: PostgreSQL
- Tool: Rate Limiter, Semaphore
- AI/ML: 딥러닝 기반 확산 예측 모델 (연구 중)
