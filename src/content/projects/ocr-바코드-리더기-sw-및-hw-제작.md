## 📖 상세 내용
프로젝트 개요
오염된 라벨의 숫자 OCR 인식 모델을 경량화 및 고도화하여 기존 바코드 처리와 동일한 속도로 분류를 수행하는 IoT 장비 개발을 위해 데이터셋 제작 및 AI 모델 구현과 장비 프로토타입 개발을 연구 개발
주요 목적
- 오염된 라벨 OCR 인식 문제 해결
- AI 모델 경량화로 IoT 장비 실시간 처리
- 바코드 대체 가능한 OCR 시스템 개발
주요 기능
- 오염 라벨 대응 OCR 인식 모델
- Data Augmentation으로 다양한 오염 상태 학습
- SBC 기반 IoT 장비 프로토타입
프로젝트 성과
- 오염된 라벨에서도 높은 인식률 달성
- 기존 바코드 처리와 동일한 속도로 분류 수행
- IoT 장비에서 실시간 처리 가능
## 🛠️ 사용 기술 및 라이브러리
- 언어: Python
- AI/ML: TensorFlow, CV, OCR
- Tool: Data Augmentation, SBC/Micro Controller

- 데이터셋 제작
- OCR 모델
### [HW 개발]
