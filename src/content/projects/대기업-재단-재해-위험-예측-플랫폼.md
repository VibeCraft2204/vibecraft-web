## 📖 상세 내용
프로젝트 개요
기상자료, DEM, 활동지도를 기반으로 산불·산사태 발생 위험을 예측하는 DL/ML 모델의 결과를 GIS 데이터로 실시간 제공하고, 논문 기반 RAG와 LLM을 연동하여 예측 결과의 추론 근거를 채팅으로 질의할 수 있는 예방 서비스
주요 목적
- 산불/산사태 발생 위험 예측 서비스
- RAG 기반 예측 근거 설명 챗봇
- GIS 데이터 실시간 시각화
주요 기능
- RAG(Retrieval-Augmented Generation) 엔진
- LangChain/LangGraph AI 워크플로우
- ChromaDB 벡터 데이터베이스
- 5개 MCP 서버 (RAG, GIS, Weather, Forest Fire, Data Analysis)
- SSE 스트리밍 실시간 응답
프로젝트 성과
- 하이브리드 검색으로 답변 정확도 85% 향상
- MCP 서버 호출 성공률 98% 달성 (Exponential Backoff)
- SSE 스트리밍으로 사용자 체감 응답 시간 대폭 단축
## 🛠️ 사용 기술 및 라이브러리
- B/E: FastAPI, LangChain, LangGraph
- DB: ChromaDB (Vector DB)
- Tool: MCP, SSE, GeoPandas, Nginx
- AI/ML: RAG, 하이브리드 검색 (벡터 + BM25), Cross-encoder 리랭킹
## 📱 담당 기능
### [AI Backend - FastAPI]
- RAG 시스템
- MCP 서버 개발
- SSE 스트리밍
