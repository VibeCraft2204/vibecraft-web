## 📖 상세 내용
프로젝트 개요
해외 여행객·교민·주재원·유학생을 대상으로, 국내 상품을 항공 기반 24시간 특송으로 전 세계 어디서나 호텔 또는 거주지에서 수령할 수 있도록 제공하는 글로벌 오픈마켓 역직구 소셜 커머스 플랫폼
주요 목적
- 글로벌 역직구 소셜 커머스 플랫폼 제공
- 해외 사용자를 위한 국내 상품 특송 서비스
- 크로스플랫폼 모바일 앱 개발
주요 기능
- Firebase Dynamic Links 딥링킹 시스템
- GetX/Provider 상태 관리
- Shimmer 로딩 효과 UI
- WebView JavaScript Bridge
- 복잡한 상품 옵션 선택 시스템
프로젝트 성과
- Shimmer 효과로 로딩 시간 체감 대폭 감소
- Dynamic Links로 바이럴 마케팅 기능 제공
- 카테고리별 랭킹 30초마다 자동 갱신
## 🛠️ 사용 기술 및 라이브러리
- F/E: Flutter (iOS/Android), Dart
- 상태관리: GetX, Provider
- B/E 연동: REST API, GraphQL
- Tool: Firebase Dynamic Links, Shimmer
