## 📖 상세 내용
프로젝트 개요
SVN + 수동 배포 환경에 GitLab CI/CD를 도입하여, Java/C#, Linux/Windows 혼재 환경에서
6개 팀 50명+ 개발자 대상 통합 파이프라인 템플릿을 설계했습니다.
주요 성과
- SVN + 수동 배포 → GitLab CI/CD 전환, 6개 팀 50명+ 대상
- Java/C#, Linux/Windows 이기종 통합 CI/CD 템플릿 설계
- Nexus 기반 공통 빌드 환경으로 개별 서버 의존성 제거
- 도입 후 1년간 파이프라인 30,000회+ 실행, 전사 표준 정착
## 🛠️ 사용 기술 및 라이브러리
- CI/CD: GitLab CI/CD, Nexus
- Build: Java (Gradle/Maven), C# (MSBuild)
- Runner: Linux/Windows 이중화

### [CI/CD 플랫폼]
- 이기종 통합 파이프라인 템플릿 개발
- Nexus 기반 공통 빌드 환경
### [전사 도입]

- 40개 프로젝트 마이그레이션
