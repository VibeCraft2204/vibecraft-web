## 📖 상세 내용

**프로젝트 개요**
탄소배출·감축 시나리오를 통합적으로 분석하여 대국민 시뮬레이션 서비스로 제공하는 탄소중립 연구 플랫폼 고도화 프로젝트

**주요 목적**
- 탄소중립 연구 플랫폼 대국민 서비스화
- 탄소배출/감축 시나리오 시뮬레이션
- 다국어 지원 CMS 구축

**주요 기능**
- Spring Boot 및 eGovFramework 기반 CMS
- JSP 기반 다국어(한국어/영어) 뷰 시스템 (131개 파일)
- Google Maps API 연동 지도 시스템
- 환경별 설정 관리 (dev, staging, production)

**프로젝트 성과**
- JSP 뷰 파일 131개 작성으로 대규모 CMS 완성
- 한국어/영어 완전 분리 관리로 국제 사용자 접근성 확보
- UUID 파일명 시스템으로 파일 충돌 100% 방지
## 🛠️ 사용 기술 및 라이브러리
- B/E: Spring Boot, eGovFramework, MyBatis
- F/E: JSP, JavaScript, HTML/CSS
- DB: MySQL
- Tool: Google Maps API, YAML 설정 관리
## 📱 담당 기능
### [Backend - Spring Boot/eGovFramework]
- CMS 백엔드
- 다국어 시스템
- 시뮬레이션 플랫폼
### [DevOps]
- 환경 관리
