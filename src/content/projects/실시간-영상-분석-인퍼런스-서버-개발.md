## 📖 상세 내용

**프로젝트 개요**
공공 치안기관의 보행 패턴 분석 시스템에서 학습된 YOLO 모델을 활용하여
실시간으로 객체 탐지 및 자세 추정을 수행하는 인퍼런스 서버를 개발했습니다.

**주요 성과**
- YOLO v5/v7 기반 실시간 객체 탐지 및 자세 추정
- Kafka 기반 분산 처리 파이프라인 (초당 30프레임)
- GPU 환경 Docker 컨테이너 기반 서빙 아키텍처
- ClearML 기반 모델 파인튜닝 실험 모니터링 (MLOps)
## 🛠️ 사용 기술 및 라이브러리
- AI: YOLOv5/v7, OpenCV, MMPose
- Pipeline: Kafka Producer/Consumer
- MLOps: ClearML
- DB: MongoDB
- Infra: Docker, GPU
## 📱 담당 기능
### [인퍼런스 엔진]
- YOLOv5/v7 서빙 환경 구현
- GPU 최적화 및 배치 처리
### [분산 처리]
- Kafka 기반 부하 분산 파이프라인
- 초당 30프레임 실시간 처리
