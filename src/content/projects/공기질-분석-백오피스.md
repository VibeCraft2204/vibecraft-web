## 📖 상세 내용

**프로젝트 개요**
스마트 공기질 모니터링과 연동된 조리실·주방 공기질 데이터를 조리 항목 기반으로 분석하여 유해물질 정보와 공기질 패턴을 자동 리포팅하고, 이상치 제거 및 기간·시간별 조회 기능을 웹으로 제공하는 공기질 분석 플랫폼

**주요 목적**
- 조리 활동과 실내 공기질 상관관계 분석 자동화
- 수작업 Excel 분석을 자동화된 파이프라인으로 대체
- 비개발자도 사용 가능한 웹 인터페이스 제공

**주요 기능**
- 6단계 데이터 분석 파이프라인 (Excel 파싱 → 메뉴 분류 → 페어 매칭 → 공기질 분석 → Before/After 비교 → 리포트 생성)
- TF-IDF 기반 메뉴 자동 분류 (8개 조리 카테고리, 144개 키워드)
- PDF/PPTX 자동 보고서 생성
- 코사인 유사도 기반 Auto-Pair 날짜 매칭
- t-test 통계 검증 및 IQR 이상치 제거

**프로젝트 성과**
- 생산성 96배 향상 (Excel 수동 분석 8시간 → 자동화 5분)
- 월 120개 보고서 자동 생성 (30개 매장 × 주간 리포트)
- Auto-Pair 매칭으로 분석 정확도 30% 향상
- 91개 파일 모듈화로 유지보수성 50% 향상
## 🛠️ 사용 기술 및 라이브러리
- F/E: React 19.2.0, TypeScript 5.9.3, Vite 7.2.4, TanStack Query, Zustand, Tailwind CSS
- B/E: FastAPI ≥0.104.0, SQLAlchemy ≥2.0.0, pandas ≥2.0.0, scipy ≥1.10.0
- DB: SQLite
- Tool: reportlab ≥4.0.0 (PDF), python-pptx ≥0.6.21 (PPTX), Playwright 1.57.0 (E2E Test)
- AI/ML: scikit-learn ≥1.3.0 (TF-IDF), 코사인 유사도

- 6단계 데이터 분석 파이프라인
- 11개 REST API 라우터
- 자동 리포트 생성

- 대시보드 UI (44개 파일)
