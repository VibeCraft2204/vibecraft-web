## 📖 상세 내용

**프로젝트 개요**
환경영향평가 연구과제에서 뉴스기사를 수집하고 DNN 기반 NLP 분류 모델과 추출 요약 모델을 구현했습니다. 텍스트 분류 정확도를 기존 5%대에서 60%로 개선한 프로젝트입니다.

**주요 성과**
- 텍스트 분류 정확도 5% → 60%로 개선

## 🛠️ 사용 기술 및 라이브러리
- ML/DL: PyTorch, HuggingFace, KoBERT, GPT2, BERT, LSTM
- Crawling: Selenium
- Platform: Linux

## 📱 담당 기능

**[데이터 수집]**
- Selenium 기반 텍스트 데이터 크롤링 모듈 구현

**[분류 모델]**
- LSTM 기반 테스트 모델 구현 및 적정성 검토
- GPT2, BERT에 Classification 레이어 추가 Fine-Tuning한 뉴스기사 분류 모델 구현

**[중복 검사]**
- TF-IDF 및 Cosine Similarity 기반 문서 간 유사도 검사 및 중복 제거 시스템 구현

**[요약 모델]**
- BERTSUM Fine-Tuning 알고리즘을 HuggingFace 기반 KoBERT에 적용한 추출 요약 모델 구현
