## 프로젝트 개요

그린마켓은 친환경 인증 제품만을 큐레이션하여 판매하는 이커머스 스타트업으로, 기존 오픈마켓 입점 방식의 수수료 부담과 브랜드 경험 제어 한계를 극복하고자 자체 플랫폼을 원했습니다. 친환경 가치를 전달하는 브랜드 톤앤매너를 유지하면서도 구매 전환까지의 경험을 최대한 매끄럽게 설계하는 것이 핵심 목표였습니다. VibeCraft는 상품 관리부터 결제, 배송 추적, 리뷰까지 이커머스 전 기능을 포함한 플랫폼을 6개월 이내에 구축했습니다.

## 도전 과제

친환경 인증 정보, 성분 데이터, 환경 영향 지표 등 일반 이커머스와 다른 복잡한 상품 속성을 유연하게 관리할 수 있는 데이터 모델 설계가 필요했습니다. 모바일 트래픽이 전체의 78%를 차지하는 특성을 고려해 Core Web Vitals 최적화가 중요한 과제였으며, Stripe를 통한 정기결제와 일시결제를 동시에 지원하는 결제 플로우 구현도 복잡성을 더했습니다.

## 접근 방식

Next.js App Router와 ISR을 활용하여 상품 상세 페이지의 SEO와 로딩 성능을 동시에 확보했고, Redis 캐싱으로 인기 상품 목록 조회를 최적화했습니다. AWS S3와 CloudFront를 통해 상품 이미지를 CDN으로 서빙하여 이미지 로딩 속도를 크게 개선했습니다. 체크아웃 단계는 사용자 이탈 데이터를 분석하여 불필요한 입력 필드를 최소화하는 방향으로 반복 개선했습니다.

## 주요 성과

플랫폼 오픈 후 4개월 만에 월 거래액 8억원을 달성하며 목표치를 조기 돌파했습니다. 체크아웃 프로세스 개선으로 전환율이 35% 향상되었고, LCP 기준 Core Web Vitals 점수가 95점 이상을 유지하고 있습니다. 재구매율도 월 평균 38%를 기록하며 충성 고객층이 빠르게 형성되고 있습니다.
