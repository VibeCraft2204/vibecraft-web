## 📖 상세 내용

**프로젝트 개요**
국가 인증기관 미세먼지 간이측정기 성능 평가 1등급 달성을 위해 시계열 데이터 전처리 방식을 설계하고 회귀 분석을 적용해 인증 통과 확률 개선하는 프로젝트로 2번의 인증 시험에서 2번 모두 1등급 달성 (100%)

**주요 목적**
- 국가 인증기관 성능 평가 1등급 인증 획득
- 시계열 데이터 전처리 방식 설계
- 회귀 분석 기반 보정 계수 산출

**주요 기능**
- 시계열 데이터 전처리 자동화
- 이상치 제거 알고리즘
- 선형 회귀 기반 보정 모델

**프로젝트 성과**
- 2번의 인증 시험에서 2번 모두 1등급 달성 (성공률 100%)
- 시계열 데이터 전처리 및 회귀 분석으로 측정 정확도 향상
## 🛠️ 사용 기술 및 라이브러리
- 언어: Python
- 데이터 분석: Pandas, NumPy, Matplotlib
- AI/ML: Scikit-Learn (선형 회귀)
