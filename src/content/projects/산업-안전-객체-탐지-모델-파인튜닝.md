## 📖 상세 내용

**프로젝트 개요**
산업 현장의 안전 위반 사항(안전모 미착용, 쓰러진 작업자, 화재)을
실시간으로 탐지하기 위해 YOLOv5 모델을 파인튜닝했습니다.

**주요 성과**
- 안전모 미착용/쓰러진 사람/화재 3종 탐지 모델 개발
- 수만 장 이미지 데이터셋 기반 파인튜닝
- ClearML 기반 실험 관리 및 모델 버전 관리
- 특정 CCTV 환경 최적화로 실제 배포 성능 극대화
## 🛠️ 사용 기술 및 라이브러리
- AI: YOLOv5, PyTorch
- MLOps: ClearML
- GPU: RTX 3090
## 📱 담당 기능
### [모델 파인튜닝]
- YOLOv5 전이학습
- 하이퍼파라미터 튜닝
- RTX 3090 GPU 학습 최적화
### [실험 관리]
- ClearML 실험 추적/비교
- 모델 체크포인트 관리
