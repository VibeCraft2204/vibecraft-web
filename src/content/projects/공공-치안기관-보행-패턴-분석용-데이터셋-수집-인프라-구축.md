## 📖 상세 내용

**프로젝트 개요**
공공 치안기관의 보행 패턴 분석 연구를 위해 CCTV 80대가 둘러싼 무대 환경에서
보행 데이터를 수집하는 인프라를 구축했습니다.

**주요 성과**
- CCTV 80대 규모 무대 환경 구성 (네트워크 설정 포함)
- 에어갭 환경 디스크 기반 데이터 이동 체계 구축
- 자동 프레임 분할 및 관절 라벨링 도구 개발
- 학습용 데이터셋 자동 생성 파이프라인 구현
## 🛠️ 사용 기술 및 라이브러리
- Tool: C#/WPF (관절 라벨링 GUI)
- AI: YOLO, MMPose
- Infra: Python, OpenCV
## 📱 담당 기능
### [무대 환경 구성]
- CCTV 80대 설치 및 네트워크 구성
- 에어갭 보안 체계
### [라벨링 도구]
- C#/WPF 관절 라벨링 GUI 개발
- YOLO/MMPose 형식 데이터셋 자동 생성
