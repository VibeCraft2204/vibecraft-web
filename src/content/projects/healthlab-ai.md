## 프로젝트 개요

헬스랩은 영상의학과 전문의와 협력하여 흉부 X-ray 영상의 이상 소견을 AI가 선별하고 우선순위를 제안하는 진단 보조 시스템을 개발하고자 했습니다. 영상의학과 의사 1인이 하루 처리해야 할 판독 건수가 꾸준히 증가하는 상황에서, AI가 경증·정상 케이스를 1차로 필터링하여 의사가 복잡한 케이스에 집중할 수 있는 환경을 만드는 것이 핵심 목표였습니다. VibeCraft는 모델 학습부터 임상 연동 API까지 전체 파이프라인을 구축했습니다.

## 도전 과제

학습에 사용할 수 있는 레이블링된 의료 영상 데이터 확보가 가장 큰 제약이었으며, 데이터 부족 문제를 해결하기 위한 전략이 필요했습니다. 의료 AI 특성상 오탐(False Negative)의 리스크가 크기 때문에, 정밀도보다 재현율을 우선하는 모델 튜닝 방향이 요구되었습니다. GCP 환경에서 HIPAA에 준하는 의료 데이터 보안 기준을 충족하는 인프라 설계도 필수 조건이었습니다.

## 접근 방식

공개 의료 영상 데이터셋(ChestX-ray14)으로 PyTorch 기반 EfficientNet 모델을 사전 학습하고, 헬스랩이 보유한 내부 데이터로 파인튜닝하는 전이학습 전략을 적용했습니다. 데이터 증강(augmentation) 기법으로 학습 데이터를 확장하고, 앙상블 모델 구조로 단일 모델 대비 오탐율을 낮췄습니다. FastAPI 기반의 추론 서버를 Docker 컨테이너로 패키징하여 병원 EMR 시스템과의 연동 API를 표준화했습니다.

## 주요 성과

임상 검증 결과 진단 보조 정확도 94.2%를 달성하며 기존 목표치(90%)를 초과했습니다. AI 필터링 도입 후 의사의 판독 처리 시간이 70% 감소했으며, 의사들은 복잡한 케이스 검토에 더 많은 시간을 투입할 수 있게 되었습니다. 현재 3개 협력 병원에서 파일럿 운영 중이며, 식약처 소프트웨어 의료기기 인허가를 준비하고 있습니다.
