## 📖 상세 내용

**프로젝트 개요**
환경영향평가 연구과제의 데이터 자동수집·적재 시스템을 1인 풀스택으로 설계·구현했습니다. REST API, 웹 크롤링, DB, 문서파일 등 다양한 수집 매체를 지원하는 소규모 ETL 플랫폼입니다.

**주요 성과**
- 1인 개발로 요구사항 정의부터 프로토타입 구현까지 전 과정 수행

## 🛠️ 사용 기술 및 라이브러리
- Frontend: React (JS)
- Backend: Spring Boot (Java), FastAPI (Python)
- DB: PostgreSQL, MongoDB
- Infra: Linux, Nginx, Tomcat
- ETL: Luigi

## 📱 담당 기능

**[Frontend]**
- ReactJS 기반 REST JSON 데이터 송수신 및 화면 표출

**[Web/WAS 운영·제어]**
- Nginx, Tomcat, Spring Boot 기반 데이터 수집·적재 운영정보 관리 모듈 MSA 설계/구현

**[데이터 수집·적재]**
- Open API / 웹 크롤링 데이터 수집·적재 및 스케줄링 모듈 MSA 설계/구현
- ETL 모니터링 프레임워크(Luigi) 적용
