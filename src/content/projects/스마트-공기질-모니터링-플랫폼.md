## 📖 상세 내용
프로젝트 개요
다중이용시설에 설치된 자사 장비에서 측정된 8가지의 항목을 기반으로 기저질환 및 공기질 민감군이 자발적으로 실내 공기질 관리를 수행하도록 보조하는 실내 공기질 모니터링 앱 플랫폼
주요 목적
- 실시간 실내 공기질 모니터링
- 공기질 기준 초과 시 알림 제공
- 다중이용시설 공기질 규제 대응
주요 기능
- 10개 센서 기반 통합 공기질 지수(IAQ) 계산
- FCM 푸시 알림 (멀티캐스트 최대 500개)
- Flutter 모바일 앱 (실시간 차트)
- JWT 토큰 인증 시스템
- 다국어 지원 (한국어/영어/일본어)
프로젝트 성과
- FCM 푸시 알림 성공률 98% 달성
- 동시 접속자 500명 처리 성능 검증
- 플레이스토어/앱스토어 정식 배포
- 100개 이상 시설에 설치 운영
## 🛠️ 사용 기술 및 라이브러리
- F/E: Flutter 3.10, Dart, Provider, GetX
- B/E: FastAPI, SQLAlchemy, JWT
- DB: MySQL 8.0, Elasticsearch 7.x
- Tool: Docker, GitHub Actions, Nginx, Rocky Linux, Prometheus
- AI/ML: LSTM (시계열 예측), Scikit-Learn
