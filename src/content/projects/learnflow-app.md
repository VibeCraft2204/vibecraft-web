## 프로젝트 개요

런플로우는 바쁜 직장인을 위해 하루 10분 이내의 짧은 강의 단위로 직무 스킬을 학습하는 마이크로러닝 플랫폼입니다. 기존 웹 기반 서비스는 모바일 접근성이 낮고 학습 연속성이 끊기는 문제로 완주율이 20% 수준에 머물렀습니다. VibeCraft는 오프라인 지원, 스트릭 시스템, 개인화 추천을 핵심 기능으로 탑재한 네이티브 모바일 앱을 개발하여 학습 지속성을 높였습니다.

## 도전 과제

지하철이나 오프라인 환경에서도 끊김 없이 학습을 이어갈 수 있는 오프라인 우선 아키텍처 설계가 핵심 기술 과제였습니다. 사용자별 학습 패턴과 직무 분야를 고려한 콘텐츠 추천 로직을 앱 단에서 가볍게 처리해야 했으며, React Native와 Expo의 성능 한계를 극복하는 최적화 작업도 필요했습니다. 애니메이션과 인터랙션이 많은 게임화 UI 구현에서 60fps를 유지하는 것도 중요한 품질 기준이었습니다.

## 접근 방식

Supabase의 실시간 동기화와 로컬 SQLite를 결합한 오프라인 우선 데이터 전략을 구현하여, 네트워크 복구 시 자동으로 학습 기록을 동기화하도록 설계했습니다. Expo의 네이티브 모듈을 활용해 하드웨어 가속 애니메이션을 적용하고, 무거운 리스트 렌더링에는 FlashList를 도입하여 스크롤 성능을 개선했습니다. 학습 스트릭, 배지 시스템, 주간 챌린지 등의 게임화 요소를 사용자 행동 데이터 분석 결과를 기반으로 우선순위를 정해 순차 도입했습니다.

## 주요 성과

앱 출시 4개월 만에 등록 사용자 8,500명을 돌파했으며, 강의 완주율이 기존 웹 대비 42% 향상된 62%를 기록했습니다. 앱 설치 후 7일 유지율이 업계 평균 대비 18% 포인트 높게 나타났으며, 일평균 학습 세션 시간은 14분으로 목표치인 10분을 초과 달성했습니다. 구글 플레이 교육 카테고리 신규 인기 앱 섹션에 2주간 선정되었습니다.
