## 프로젝트 개요

SYC는 교육기관과 기업 간의 자격 증명 발급 및 검증 프로세스를 블록체인 기반 DID 기술로 디지털화하는 플랫폼을 개발하고자 했습니다. 기존의 종이 증명서 방식은 위변조 리스크와 긴 처리 시간 문제를 안고 있었으며, 수요자와 공급자 모두를 위한 통합 웹 플랫폼이 필요했습니다. VibeCraft는 발급 기관 관리자, 자격 증명 보유자, 검증 기관을 위한 세 가지 역할 기반 인터페이스를 포함한 풀스택 플랫폼을 구축했습니다.

## 도전 과제

블록체인 트랜잭션의 특성상 응답 지연이 발생하는 구간을 사용자 경험 관점에서 어떻게 처리할지가 핵심 과제였습니다. 또한 기관별로 상이한 증명서 양식과 필드 구조를 유연하게 수용할 수 있는 데이터 모델 설계가 필요했습니다. AWS 인프라 위에 Docker 컨테이너를 배포하면서 무중단 배포와 보안 정책을 동시에 충족하는 것도 중요한 도전이었습니다.

## 접근 방식

Next.js App Router 기반으로 서버 컴포넌트와 클라이언트 컴포넌트를 역할에 맞게 분리하여 초기 렌더링 성능을 최적화했습니다. Prisma를 ORM으로 채택하여 복잡한 다대다 관계를 가진 증명서 스키마를 타입 안전하게 관리했으며, PostgreSQL의 JSON 타입을 활용해 기관별 커스텀 필드를 유연하게 저장했습니다. 블록체인 트랜잭션은 비동기 큐로 처리하고 상태를 실시간으로 폴링하여 사용자가 진행 상황을 즉시 확인할 수 있도록 설계했습니다.

## 주요 성과

50개 이상의 REST API 엔드포인트를 구축하고, 자격 증명 발급 및 검증 요청의 평균 처리 시간을 200ms 이하로 달성했습니다. 베타 런칭 후 3개 교육기관이 플랫폼에 온보딩하여 실제 증명서 발급 운영을 시작했으며, 위변조 검증 정확도는 100%를 기록했습니다. AWS 기반 인프라의 가용성은 99.9% 이상을 유지하고 있습니다.
