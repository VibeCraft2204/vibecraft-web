## 📖 상세 내용

**프로젝트 개요**
전국 물계정 정보를 대국민 서비스 형태로 제공하고, 복잡한 산출 지표를 자동 계산·시각화하는 공공 물계정 분석 플랫폼

**주요 목적**
- 전국 물계정 정보 대국민 서비스 제공
- PSUT(Physical Supply and Use Table) 자동 계산
- 복잡한 지표 자동 산출 및 시각화

**주요 기능**
- PSUT 자동 계산 시스템 (RAS 방법 균형 조정)
- Elasticsearch 통합 검색 시스템
- JWT 토큰 기반 인증 시스템
- Docker Compose 마이크로서비스 아키텍처

**프로젝트 성과**
- PSUT 처리 시간 90% 단축, 물 자원 관리 효율 300% 향상
- 대용량 데이터 검색 속도 10배 향상 (Elasticsearch)
- 복잡한 쿼리 0.1초 이내 응답 달성
## 🛠️ 사용 기술 및 라이브러리
- B/E: FastAPI (PSUT 계산), Spring Boot (웹 서비스), eGovFramework
- DB: PostgreSQL, MySQL, Elasticsearch
- Tool: Docker Compose, Nginx, JWT
