## 📖 상세 내용

**프로젝트 개요**
LAN을 통한 실시간 데이터 송수신 및 데이터 파싱을 수행하고 DSP 응용 기능을 제공하는 Spectrogram 모니터링 툴 제작

**주요 목적**
- 실시간 DAQ 데이터 모니터링
- DSP 응용 Spectrogram 분석
- 윈도우 데스크탑 어플리케이션 제공

**주요 기능**
- LAN 기반 실시간 데이터 송수신
- 데이터 파싱 엔진
- DSP Spectrogram 시각화
- MFC 기반 윈도우 UI

**프로젝트 성과**
- LAN 기반 실시간 데이터 수신 및 시각화
- DSP 응용 Spectrogram 분석 기능 제공
