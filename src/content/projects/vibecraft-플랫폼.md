## 📖 상세 내용

**프로젝트 개요**
민간·기업·공공 데이터를 RAG와 MCP 기반 AI 에이전트로 자동 통합·분석하여 소상공인의 매출, 상권, 운영, 정책 지원까지 원샷 대시보드와 대화형 질의응답으로 제공하는 AI 경영 지원 플랫폼

**주요 목적**
- 민간/기업/공공 데이터 자동 통합 및 분석
- 소상공인 맞춤형 AI 경영 지원
- 대화형 질의응답 시스템 제공

**주요 기능**
- LangGraph 3단계 워크플로우 (분석-생성-통합)
- RAG 기반 데이터 자동 통합 및 분석
- MCP 서버를 통한 다양한 데이터 소스 연동
- Gemini API 연동 LLM 시스템
- SSE 스트리밍 기반 실시간 진행 상황 전송

**프로젝트 성과**
- 요구사항만으로 완전한 React 앱 자동 생성
- 3단계 워크플로우로 구조화된 코드 품질 보장
- Windows/Unix 크로스 플랫폼 지원
## 🛠️ 사용 기술 및 라이브러리
- F/E: React, TypeScript
- B/E: FastAPI, LangChain, LangGraph, Gemini API
- DB: ChromaDB (Vector DB)
- Tool: MCP (Model Context Protocol), SSE, Docker
- AI/ML: RAG, LangChain, LangGraph, Gemini API
