프로젝트 개요

주요 목적
- 개발자를 위한 경량화된 AI 분석 도구 제공
- 명령줄 환경에서의 빠른 데이터 분석
- Vibecraft 플랫폼의 CLI 버전 제공으로 사용자층 확대
주요 기능
- CLI 기반 AI 에이전트 인터페이스

- 자연어 기반 데이터 질의
