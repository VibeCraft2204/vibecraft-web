## 📖 상세 내용

**프로젝트 개요**
소셜 커머스 관리자/판매자를 위한 상품, 기획전, 매출, 배송 관리까지 한 번에 가능한 웹 어드민 플랫폼

**주요 목적**
- 관리자/판매자 통합 대시보드 제공
- 상품, 주문, 배송 관리 효율화
- 실시간 데이터 동기화

**주요 기능**
- 관리자 대시보드 (판매 통계, 사용자 관리)
- 판매자 관리 시스템 (상품 등록, 재고, 주문)
- 매출 분석 및 시각화
- 데이터 엑스포트 기능

**프로젝트 성과**
- Flutter Web 기반 통합 관리 시스템 구현
- 관리자/판매자 직관적 UI/UX 제공
- 실시간 데이터 동기화 및 알림 시스템 구축
## 🛠️ 사용 기술 및 라이브러리
- F/E: Flutter Web
- 상태관리: GetX, Provider
- UI: Material Design, 반응형 웹
- B/E 연동: RESTful API
## 📱 담당 기능
### [Flutter Web - Admin]
- 관리자 대시보드
- 판매자 관리
- 리포팅
