## 프로젝트 개요

핀페이는 소상공인을 대상으로 하는 간편결제 서비스로, 서비스 성장과 함께 거래량이 급증하면서 기존의 엑셀 기반 정산 관리와 수동 모니터링 방식의 한계에 부딪혔습니다. 실시간 거래 현황, 오류 감지, 가맹점별 정산 내역을 한 화면에서 확인하고 즉각 대응할 수 있는 운영 도구가 필요했습니다. VibeCraft는 핀테크 도메인의 복잡한 데이터 구조를 시각화하는 어드민 대시보드를 처음부터 설계하고 구축했습니다.

## 도전 과제

일 50만 건에 달하는 거래 데이터를 실시간으로 집계하고 시각화하는 과정에서 데이터베이스 쿼리 성능이 병목으로 작용했습니다. 또한 금융 서비스 특성상 데이터 접근 권한 관리가 세분화되어야 했으며, 감사 로그를 빠짐없이 기록하는 컴플라이언스 요구사항도 충족해야 했습니다. Chart.js로 구현한 대용량 시계열 차트의 렌더링 성능 최적화도 주요 기술 과제였습니다.

## 접근 방식

Next.js의 서버 컴포넌트를 활용하여 초기 페이지 로드 시 집계 데이터를 서버에서 미리 처리하고, 실시간 업데이트가 필요한 영역만 클라이언트에서 폴링하도록 구조를 분리했습니다. Prisma의 원시 쿼리 기능을 활용해 복잡한 집계 쿼리를 최적화하고, 자주 조회되는 정산 데이터는 캐싱 레이어를 도입했습니다. 역할 기반 접근 제어(RBAC)를 미들웨어 레벨에서 구현하여 권한 관리의 일관성을 확보했습니다.

## 주요 성과

24/7 무중단 실시간 모니터링 환경을 구축하여 이상 거래 탐지 시간을 평균 4시간에서 10분 이내로 단축했습니다. 정산 담당자의 월말 정산 작업 시간이 3일에서 반나절로 줄었으며, 수작업 오류도 사실상 사라졌습니다. Vercel 배포 환경에서 99.95%의 가동률을 유지하며 안정적인 서비스 운영을 지원하고 있습니다.
