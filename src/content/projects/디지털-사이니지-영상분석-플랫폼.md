## 📖 상세 내용

**프로젝트 개요**
기존 비개발자가 바이브코딩으로 Ruby를 이용해 구현한 Legacy 프로젝트를 마이그레이션 및 리펙토링하고, 관공서 공지·공익광고의 디지털 전환을 통해 시민 시청 행위를 객체 탐지·Pose Estimation으로 비식별 측정하고, QR 유입 데이터를 KPI로 분석하여 광고 효과와 디지털 사이니지 최적 배치를 모니터링하는 스마트 디지털 사이니지 플랫폼

**주요 목적**
- 관공서 디지털 사이니지 광고 효과 측정
- 비식별 시청자 행동 분석 (객체 탐지, Pose Estimation)
- QR 코드 기반 유입 데이터 KPI 분석

**주요 기능**
- OpenCV 기반 비디오/이미지 QR 코드 실시간 처리
- Spring Boot 백엔드 API 서버
- JPA/Hibernate 데이터베이스 최적화
- S3 스토리지 연동 파일 관리
- Analytics 대용량 데이터 배치 처리

**프로젝트 성과**
- QR 코드 추출 처리 속도 70% 향상 (디스크 I/O 완전 제거)
- 데이터베이스 부하 60% 감소 (PathVariable Entity Converter 패턴)
- 컨트롤러 코드 30% 라인 감소
- I/O 대기 시간 100% 제거
## 🛠️ 사용 기술 및 라이브러리
- F/E: Android (Kotlin)
- B/E: Spring Boot, JPA/Hibernate, OpenCV
- DB: MySQL/PostgreSQL
- Tool: Docker, Docker Compose, GitHub Actions CI/CD, S3
- AI/ML: OpenCV (QR 코드 감지), 객체 탐지, Pose Estimation
