## 📖 상세 내용

**배경 및 개요**
- 매 반기마다 수작업으로 진행되는 대규모 사업자 정보 갱신 업무로 인한 데이터 오기입 위험 및 업무 효율성 개선

**주요 수행 내용**
- Unix Cron Job을 활용하여 지정된 스케줄마다 대용량 텍스트 파일을 자동으로 파싱하고 DB에 적재하는 일괄 처리 배치 프로그램 개발
- 배치 처리 과정에서 발생하는 OOM 현상 방지를 위해 임계치까지 데이터를 버퍼링 후 DB에 일괄 반영하는 메모리 최적화 로직 적용
- 사업자 정보 갱신 파일의 업로드 처리 현황(성공/실패) 및 데이터 갱신 실패 사유를 모니터링할 수 있는 통합 관리 백오피스 구축

**주요 성과**
- 텍스트 파일 기반 대용량 데이터 처리 배치 프로그램 및 통합 관리 백오피스 구축을 통한 업무 프로세스 완전 자동화
- 기존 수작업 대비 데이터 갱신 업무에 소요되는 시간 90% 이상 단축 및 휴먼 에러로 인한 데이터 오기입 발생률 0% 달성

## 🛠️ 사용 기술
- Backend: Struts, Spring Framework, MyBatis, Cron Job (Unix)
- Frontend: JSP, Javascript, jQuery
- Database: Oracle
