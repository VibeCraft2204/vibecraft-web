## 프로젝트 개요

K공공기관은 정책 수립과 성과 관리에 필요한 데이터가 여러 부처 시스템과 외부 기관에 분산되어 있어, 담당자들이 매월 데이터를 수동으로 취합하고 엑셀로 리포트를 작성하는 비효율이 반복되고 있었습니다. 데이터를 기반으로 한 신속한 의사결정을 지원하기 위해 통합 분석 포털 구축을 의뢰했습니다. VibeCraft는 데이터 수집 파이프라인 설계부터 시각화 대시보드 구축, 리포트 자동화까지 전 과정을 컨설팅하고 구현했습니다.

## 도전 과제

47개 데이터 소스의 포맷과 갱신 주기가 모두 달라, 표준화된 수집 및 정제 파이프라인을 설계하는 것이 가장 큰 기술 과제였습니다. 공공기관 특성상 보안 심의를 통과한 오픈소스 도구만 사용해야 하는 제약이 있었으며, 데이터 분석에 익숙하지 않은 실무 담당자들도 직접 리포트를 생성할 수 있어야 했습니다. 이전 시스템과의 데이터 연속성을 유지하면서 단계적으로 전환하는 마이그레이션 계획도 수립해야 했습니다.

## 접근 방식

Airflow DAG를 소스별로 독립적으로 설계하여 특정 소스의 오류가 전체 파이프라인에 영향을 주지 않도록 격리했습니다. Pandas 기반의 데이터 정제 모듈을 공통화하여 재사용성을 높이고, PostgreSQL에 정규화된 데이터 웨어하우스 스키마를 구성했습니다. Apache Superset을 대시보드 및 리포트 도구로 채택하여 담당자들이 SQL 없이도 차트를 생성하고 공유할 수 있는 셀프서비스 환경을 제공했습니다.

## 주요 성과

47개 데이터 소스가 단일 플랫폼에 통합되었고, 월 120건의 정기 리포트가 자동으로 생성·배포되기 시작했습니다. 담당자 1인당 리포트 작업 시간이 월 40시간에서 5시간 이내로 줄었으며, 데이터 기반 회의 자료 준비 속도가 크게 빨라졌습니다. 시스템 오픈 후 기관 내 타 부서에서도 도입 문의가 이어지고 있습니다.
