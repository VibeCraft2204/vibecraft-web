## 📖 상세 내용
프로젝트 개요
공공데이터 API와 IoT 센서를 연동하여 농업 용수를 효율적으로 관리하는
백엔드 시스템을 개발했습니다.
주요 성과
- Strategy 패턴 기반 동적 수집 대상 설정 구조
- 공공데이터 API 연동 크롤링 시스템 개발
- IoT 센서 데이터 수집 백엔드 서버 개발
- Docker 컨테이너 기반 마이크로서비스 구조
## 🛠️ 사용 기술 및 라이브러리
- B/E: C#, .NET Core
- DB: MariaDB
- Infra: Docker
- IoT: REST API 기반 센서 연동
## 📱 담당 기능
### [백엔드 서버]
- C# .NET Core REST API 개발
- Strategy 패턴 기반 크롤러 아키텍처
### [데이터 수집]
- 공공데이터 API 연동 (기상/수위/댐)
- IoT 센서 데이터 수신 및 저장
