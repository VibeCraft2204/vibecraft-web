## 📖 상세 내용
프로젝트 개요
HPC 서버를 활용하여 TSP 알고리즘 기반 최단 방문 경로 설정 및 이미지 프로세싱을 통한 배송 지점 탐색 운송 드론 연구 개발
주요 목적
- TSP 알고리즘 기반 최적 경로 계산
- 이미지 프로세싱 기반 배송 지점 탐색
- 자율 비행 드론 시스템 개발
주요 기능
- HPC 서버 활용 TSP 알고리즘
- Computer Vision 기반 객체 인식
- 드론 자율주행 소프트웨어
- SBC 기반 드론 제어
프로젝트 성과
- HPC 서버를 활용한 고속 경로 계산
- 다중 배송 지점 최적 경로 탐색 성공
- 이미지 프로세싱 기반 정확한 배송 지점 인식
## 🛠️ 사용 기술 및 라이브러리
- 언어: Python, C/C++
- 알고리즘: TSP (Traveling Salesman Problem)

- Tool: HPC 서버, SBC
