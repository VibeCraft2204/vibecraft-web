## 프로젝트 개요

에듀픽은 월 수강생 3,000명 규모의 온라인 직무교육 플랫폼으로, 강의 업로드, 수강생 진도 관리, 이수증 발급, 학습 리마인더 발송 등의 반복 작업에 운영팀 인력의 70%가 소요되고 있었습니다. 콘텐츠 생산과 강사 관계 관리에 집중하고 싶지만 루틴한 운영 업무가 발목을 잡는 상황이었습니다. VibeCraft는 운영 업무 전반을 자동화하는 n8n 기반 워크플로우 시스템을 설계하고 구축했습니다.

## 도전 과제

자동화 범위가 넓은 만큼 업무 흐름을 명확하게 정의하는 사전 분석 작업이 필수적이었습니다. 특히 자동화 오류가 수강생 경험에 직접 영향을 미칠 수 있어, 예외 상황 처리와 수동 개입 전환 로직을 세심하게 설계해야 했습니다. Notion 기반의 기존 운영 도구를 유지하면서 자동화 시스템과 연동하는 방식을 선택했기 때문에 API 제약과 rate limit 처리도 주요 기술 과제였습니다.

## 접근 방식

운영팀과 두 차례의 워크숍을 통해 32개의 반복 업무를 식별하고 자동화 우선순위를 결정했습니다. n8n의 노코드 워크플로우 빌더로 핵심 파이프라인을 구성하고, 복잡한 데이터 변환 로직은 Python 스크립트로 처리했습니다. OpenAI API를 활용해 수강생 학습 패턴을 분석한 맞춤형 독려 메시지를 자동 생성하고, Notion API와 Slack API를 연결하여 운영팀이 모든 현황을 한 채널에서 확인할 수 있도록 구성했습니다.

## 주요 성과

32개 워크플로우 자동화를 통해 운영팀의 반복 업무 시간이 60% 감소했으며, 담당자 2명이 줄어든 공수로 강사 개발과 콘텐츠 기획에 재투자할 수 있게 되었습니다. 이수증 발급 오류가 월 평균 15건에서 0건으로 줄었고, 학습 리마인더 자동화로 강의 완주율이 12% 포인트 향상되었습니다. 자동화 시스템 도입 이후 운영팀 만족도 설문에서 4.6점을 기록했습니다.
