## 📖 상세 내용
프로젝트 개요
GPS를 사용할 수 없는 실내 환경에서 비콘 없이 저비용 드론 자율 주행을 구사하는 연구 개발
주요 목적
- GPS 없는 실내 자율주행 구현
- 비콘 없는 저비용 위치 인식
- Computer Vision 기반 실내 내비게이션
주요 기능
- Computer Vision 기반 위치 인식
- 저비용 드론 제어 시스템
- SBC 기반 실시간 이미지 처리
- 자율주행 알고리즘
프로젝트 성과
- GPS 없는 실내 환경에서 자율주행 성공
- Computer Vision 기반 저비용 실내 위치 인식 시스템 구현
## 🛠️ 사용 기술 및 라이브러리
- 언어: Python, C/C++
- AI/ML: Computer Vision, 이미지 프로세싱
- Tool: SBC (Raspberry Pi)
